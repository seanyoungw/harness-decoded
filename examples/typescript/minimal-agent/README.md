# Level 1 — Minimal Agent (TypeScript)

Twin of `examples/python/minimal_agent`. Same harness loop and permission model.

## Setup

```bash
npm install
export ANTHROPIC_API_KEY=sk-ant-...
```

## Run

```bash
npx ts-node agent.ts "list all TODO comments in this directory"
```
